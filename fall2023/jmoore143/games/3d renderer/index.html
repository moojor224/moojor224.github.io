this is a prototype for a first-person maze game coded and rendered entirely in html, javascript, and css. that means it doesn;t use webgl or html canvases. Everything you see is an HTML element positioned with css and manipulated by javascript. Click the resume button to play.
controls:
look: move mouse
move forward/left/back/right: w/a/s/d
sprint: shift
pause: escape

options menu and quit button not currently functional
